Brief Graph Statistics Explanation

  • Number of Non-Zero Edges (courtesy Wikipedia): Total number of edges.
  • Degree Sequence (courtesy Wolfram) : A degree sequence is a monotonic nonincreasing sequence of the vertex degrees of its graph vertices.
  • Edge Weights (courtesy Wikipedia): A weighting system that imparts a value to each edge (eg: the length of a road between two nodes).
  • Clustering Coefficient (courtesy Wikipedia): In graph theory, a clustering coefficient is a measure of the degree to which nodes in a graph tend to cluster together.

Brief Graph Statistics Explanation (cont)

  • Scan Statistic i (courtesy Wikipedia): A scan statistic or window statistic is a problem relating to the clustering of randomly positioned points.
  • Eigenvalues (courtesy Wolfram): Special set of scalars associated with a linear set of equations.
  • Between-ness Centrality (courtesy Wikipedia): Betweenness centrality is an indicator of a node's centrality in a network. It is equal to the number of shortest paths from all vertices to all others that pass through that node.
